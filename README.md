## 📌 Descripción

Frontend estático para consumir la API REST del sistema clínico.

Permite gestionar:

- Pacientes
- Doctores
- Citas médicas

## 🧰 Tecnologías

- HTML5
- CSS3
- JavaScript
- Bootstrap 5
- Bootstrap Icons

## 📁 Archivos

- `index.html`: estructura visual de la aplicación.
- `styles.css`: estilos personalizados en color verde hospital.
- `app.js`: lógica CRUD y conexión con el backend.
- `validation.js`: validaciones de formularios.

## 🚀 Ejecución local

Abrir el archivo `index.html` con Live Server.

El frontend apunta localmente a:

`http://localhost:8080/api`

## ☁️ Producción

Cuando el backend esté desplegado en Render, cambiar en `app.js` esta línea:

`https://clinic-back-end.onrender.com/api`

por la URL real del backend.
